# TechDraw ExtensionInsertSquare

## Description

The <img alt="" src=images/TechDraw_ExtensionInsertSquare.svg  style="width:24px;"> **TechDraw ExtensionInsertSquare** tool inserts a square tube section symbol at the start of the dimension text.

 <img alt="" src=images/TechDraw_extensionInsertSquareExample.png  style="width:500px;">  
*On the right a dimension with the inserted symbol*

## Usage

1.  Select one or more dimensions.
2.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_ExtensionInsertSquare.svg" width=16px> [TechDraw ExtensionInsertSquare](TechDraw_ExtensionInsertSquare.md)** button.
    -   Select the **TechDraw → Extensions: Dimensions → <img src="images/TechDraw_ExtensionInsertSquare.svg" width=16px> Insert square symbol** option from the menu.
3.  The symbol is inserted in the dimension text of the selected dimensions.
