# Draft Drawing/fr


</div>

## Description


<div class="mw-translate-fuzzy">

L\'outil Dessin vous permet de placer les objets sélectionnés sur une feuille de dessin créée avec [Atelier Mise en plan](Drawing_Workbench/fr.md). S\'il n\'y a pas de [Modèles de feuille](Drawing_templates.md) dans le document, un fichier par défaut sera créé.


</div>


<div class="mw-translate-fuzzy">

Cet outil fonctionne de la même manière que l\'outil [Insérer une vue dans la page Drawing](Drawing_View/fr.md) mais il est optimisé pour les objets de l\'[Atelier Draft](Draft_Workbench/fr.md). Il peut également restituer des objets 2D avec un remplissage de surface. Il peut gérer des objets spécifiques tels que [Cote Draft](Draft_Dimension/fr.md) et [Texte Draft](Draft_Text/fr.md) que l\'outil [Insérer une vue dans la page Drawing](Drawing_View/fr.md) ne peut pas gérer.


</div>

This command is now obsolete. Use the [TechDraw Workbench](TechDraw_Workbench.md) and the [TechDraw DraftView](TechDraw_DraftView.md) command instead.

## Utilisation


<div class="mw-translate-fuzzy">

1.  Sélectionnez les objets que vous souhaitez placer sur une feuille de dessin.
2.  Dans la même sélection, ajoutez l\'objet de page [Atelier Dessin](Drawing_Workbench/fr.md) dans lequel vous voulez dessiner la projection des objets.
3.  Appuyez sur le bouton **<img src="images/Draft_Drawing.svg" width=16px> [Dessin Draft](Draft_Drawing/fr.md)**.


</div>
