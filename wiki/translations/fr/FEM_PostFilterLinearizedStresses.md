# FEM PostFilterLinearizedStresses/fr

## Description

Crée un graphique de linéarisation des contraintes.

## Utilisation

1.  Sélectionnez un [Filtre d\'écrêtage selon une ligne](FEM_PostFilterDataAlongLine/fr.md) précédemment créé.
2.  Lancez la commande de l\'une des façons suivantes :
    -   Appuyez sur le bouton **<img src="images/FEM_PostFilterLinearizedStresses.svg" width=16px> [Graphique de linéarisation des contraintes](FEM_PostFilterLinearizedStresses/fr.md)**.
    -   Sélectionnez l\'option **Résultats → <img src="images/FEM_PostFilterLinearizedStresses.svg" width=16px> Graphique de linéarisation des contraintes** dans le menu.
3.  Un tracé XY avec les valeurs de contraintes linéarisées (membrane, membrane+flexion et total) le long de la ligne sera créé dans une fenêtre séparée.
