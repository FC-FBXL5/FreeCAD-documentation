# Arch CompPanel/fr

## Description

Groupement de plusieurs outils en relation avec les panneaux.

## Types

-   <img alt="" src=images/Arch_Panel.svg  style="width:32px;"> [Panneaux](Arch_Panel/fr.md): crée un objet de panneau à partir d\'un objet 2D sélectionné. {{Version/fr|0.15}}
-   <img alt="" src=images/Arch_Panel_Cut.svg  style="width:32px;"> [Découpe de panneaux](Arch_Panel_Cut/fr.md): crée une vue en coupe 2D à partir d\'un [Arch panneau](Arch_Panel/fr.md). {{Version/fr|0.17}}
-   <img alt="" src=images/Arch_Panel_Sheet.svg  style="width:32px;"> [Panneau de feuille](Arch_Panel_Sheet/fr.md): crée une feuille de découpe 2D comprenant des [découpes de panneau](Arch_Panel_Cut/fr.md) ou d\'autres objets 2D. {{Version/fr|0.17}}
-   <img alt="" src=images/Arch_Nest.svg  style="width:32px;"> [Calepinage](Arch_Nest/fr.md): permet d\'imbriquer plusieurs objets plats dans une forme de conteneur. {{Version/fr|0.17}}
