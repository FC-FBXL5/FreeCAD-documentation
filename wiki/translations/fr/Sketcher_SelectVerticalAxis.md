## Description

Sélectionne l\'axe vertical d\'une esquisse.

## Utilisation

1.  Entrez le mode d\'édition de l\'esquisse.
2.  Choisir {{MenuCommand|Sketch → Outils d'esquisse → <img src=images/Sketcher_SelectVerticalAxis.png style="width:16px"> Sélectionner l'axe vertical}} dans le menu principal.
3.  L\'axe vertical de l\'esquisse sera sélectionnée.
