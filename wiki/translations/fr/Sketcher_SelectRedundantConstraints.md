## Description

Sélectionne les contraintes redondantes d\'une esquisse.

## Utilisation

1.  Entrer le mode d\'édition de l\'esquisse.
2.  Choisir **Sketch → Sketcher tools → <img src=images/Sketcher_SelectRedundantConstraints.svg style="width:16px"> Sélectionner les contraintes redondantes** dans le menu principal.
3.  Les contraintes redondantes de l\'esquisse seront sélectionnées.
