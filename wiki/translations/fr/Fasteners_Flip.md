# Fasteners Flip/fr

## Description

La commande <img alt="" src=images/Fasteners_Flip.svg  style="width:24px;"> **Fasteners Flip** inverse l\'orientation des fixations sélectionnées en modifiant leur propriété **invert**.

## Utilisation

1.  Sélectionnez une ou plusieurs fixations.
2.  Il existe plusieurs façons de lancer la commande :
    -   Appuyez sur le bouton **<img src="images/Fasteners_Flip.svg" width=16px> [Fasteners Flip](Fasteners_Flip.md)**.
    -   Sélectionnez l\'option **Fasteners → <img src="images/Fasteners_Flip.svg" width=16px> Invert fastener** dans le menu.
3.  L\'orientation des fixations sélectionnées est inversée.
