FreeCAD 使用 [Qt 框架](https://en.wikipedia.org/wiki/Qt_(software))绘制和管理其界面。 该框架广泛用于很多应用程序，因此 FreeCAD 的界面非常经典，没有特别的难以理解之处。大多数按钮都是标准按钮，可以在您希望的位置找到，例如：**文件→打开，编辑→粘贴**，等等。这是FreeCAD 第一次打开时的外观，安装后，显示启动中心。

![](images/FreeCAD-v0-18-FirstStart.png )

启动中心是一个方便的"欢迎屏幕"，为新手提供有用的信息，例如您一直在处理的最新文件，FreeCAD 世界中的新功能，或有关最常见工作台的快捷信息。如果有新的稳定的FreeCAD 版本，它也会通知您。

过了一段时间，当您更熟悉 FreeCAD 时，您可能已经对首选项进行了更改，因此当FreeCAD 启动时，您会直接进入一个工作台中看到自己打开的新文档。如果你还没有更改首选项，你只需关闭起始页面选项卡并创建一个新文档。

![](images/FreeCAD-v0-18-NewProject.png )

### 工作台

请注意，上面两个屏幕截图之间的某些图标已经有所不同。这是 FreeCAD 界面中使用的最重要概念发挥作用的地方：工作台。

工作台是按专业组合在一起的工具组（工具栏按钮，菜单和其他界面控件）。想像一个你可以让不同的人一起工作的工作坊：一个人做金属工作，另一个做木材工作。每个人都在他们的工作室里有一张单独的桌子，上面有他/她工作用的特定工具。虽然工作内容不同，他们的工作对象却可以相同。类似地，FreeCAD 就是如此。

FreeCAD 界面最重要的控件是工作台选择器，你可以使用它从一个工作台切换到另一个工作台：

![](images/FreeCAD-v0-18-WorkbenchMenu.png )

新用户经常对工作台感到困惑，因为要知道在哪个工作台中查找特定工具并不总是很容易。 但是他们很快就学会了，过了一会儿就会感觉很自然------意识到这是组织 FreeCAD 提供的众多工具的很便捷的方式。工作台也可完全自定义（见下文）。相同的工具可能出现在多个工作台中。无论出现在哪个工作台，一个工具的按钮图标始终都是相同的。

在本手册的后面部分，你还会看到一个显示所有工作台内容的表格。

### 界面

让我们更好地了解界面的不同部分。

![](images/FreeCAD-v0-18-Cube.png )

-   **3D 视图**是界面的主要组成部分; 它是你正在处理的对象被绘制和操纵的地方。 你可以有同一文档（或相同对象）的多个视图，或同时打开多个文档。这些视图中的每一个都可以从主窗口中独立出来。您可以通过单击来选择对象或对象的一部分，使用鼠标按钮可以平移，缩放和旋转视图。下一章将进一步解释导航的话题。

除 3D 视图面板外，FreeCAD 还提供了以下信息面板。通过从**视图→面板**中选择它们，可以使它们可见或隐藏。显示时，面板的名称将显示在面板的左上角：

-   **组合视图**有两个选项卡：
    -   "模型"选项卡上半部份显示文档的内容和结构，下半部份显示所选对象的属性（或参数）。这些属性分为两类：
        -   数据（与几何体本身有关的属性）
        -   视图（影响几何体在屏幕上的显示方式的属性）。
    -   "任务"选项卡是在使用特定于当前使用的工作台和工具时，FreeCAD 提示您提供相关的值的位置。例如，在使用 [Draft Workbench Line Tool](Draft_Line.md) 时输入\'length\'值。在按下确定（或取消）按钮后，"任务"选项卡将自动清空显示内容，切换回"模型"选项卡。双击"模型"选项卡中的相关对象通常会重新打开相应的"任务"选项卡以修改设置。

"任务"选项卡有时会产生令人费解和令人沮丧的副作用。如果"任务"选项卡不是空白，则某些 FreeCAD 操作将无法按预期工作。例如，如果模型中有单个对象（比如长方体），双击它将打"任务"选项卡，允许你修改表征长方体的参数。如果你同时打开着["选择"视图](#Selection_view/zh-cn.md)，就会看到其中列出的长方体的内部名称。整个长方体将在 3D 视图中变为绿色，表示已选择整个长方体。单击背景将取消选择整个长方体并清除"选择"视图。目前为止，一切正常。但是，如果现在单击长方体的一个面，而不是选择该面，则不会选择任何内容，因为"任务"选项卡尚未完成。即使你没有对参数进行任何修改，FreeCAD 也会等待单击"任务"选项卡中的 **确定**（或其他）按钮。


</div>

-   **报告视图**通常是隐藏的，但是你最好打开它，因为它会列出所有信息、警告或错误，帮你了解（或调试）您可能做错的事情。
-   默认情况下，**Python 控制台**也是隐藏的。 你可以使用 Python 语言与文档内容进行交互。由于你在 FreeCAD 界面上执行的每个操作实际上都执行了一段 Python 代码，因此打开Python 控制台可以让您实时查看代码------让您在使用 FreeCAD 的途中，无意间简单地学到一点 Python。
-   **树视图**仅显示组合视图中"模型"选项卡的对象树。它通常是隐藏的。
-   **属性视图**仅显示组合视图底部显示的对象属性信息。它通常是隐藏的。
-   **选择视图**显示当前所选对象的名称。它们是工作台操作将应用的对象。在应用工作台操作之前，取消选择其中一些对象，便是用它来优化了选择。选择视图还可用于按名称搜索对象，然后选择它们。默认情况下，选择视图是隐藏的。虽然你通常可以通过查看组合视图的"模型"选项卡中的对象树来确定当前选定的对象，但对于需要选择多个对像的复杂操作，以及难以选择的情况，最好使此视图可见，这样您就可以查看名称标签，并数一数所选择的对象。

![](images/FreeCAD-v0-18-ExtrudeTask.png )

### 自定义界面

FreeCAD 的界面可深度自定义。所有面板和工具栏都可以移动到不同的位置，或者将一个堆叠在另一个上面。 在需要时，从"视图"菜单中，或者右键单击界面的空白区域，它们可以关闭，也可以重新打开。不仅如此，还有更多选项可用，例如，使用任何工作台中的工具创建自定义工具栏，或者分配和更改键盘快捷键。

这些高级自定义选项可从**工具→自定义**菜单中获得。

![](images/FreeCAD-v0-18-CustomizeInterface.png )
