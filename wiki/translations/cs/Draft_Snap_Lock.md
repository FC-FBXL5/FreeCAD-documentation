## Usage

For general information about snapping see [Draft Snap](Draft_Snap.md).

1.  To invoke **Draft Snap Lock** do one of the following   *
    -   Press the **<img src="images/Draft_Snap_Lock.svg" width=16px>** button in the Draft Snap toolbar.
    -   Press the **<img src="images/Draft_Snap_Lock.svg" width=16px><img src="images/Toolbar_flyout_arrow.svg" width=8px>** button in the [Draft snap widget](Draft_snap_widget.md) and in the menu select the **<img src="images/Draft_Snap_Lock.svg" width=16px> Snap Lock** option.
    -   Use the \"in-command\" shortcut   * **S**. This shortcut can only be used if a command is active and does not work for all commands.
