# Path Copy/pt-br

## Description

This tool creates a copy of a selected path

## Usage

1.  Select a <img alt="" src=images/Workbench_Path.svg  style="width:24px;"> [Path](Path_Workbench.md) object
2.  Invoke the Path Copy command several ways:
    -   Press the **<img src="images/Path_Copy.svg" width=24px> [Copy](Path_Copy.md)** button in the Path toolbar
    -   Use the **P** then **Y** keyboard shortcut
    -   Use the **Path → Path Modification → Copy** entry in the Path menu

## Properties

-    **Base**: The base path to be copied

## Options

The copy stays linked with the original path. If the original changes, so does the copy.
