## Prerequisites

To use BOLTS for FreeCAD

-   FreeCAD (http://freecadweb.org/)
-   python 2.6 or 2.7
-   pyyaml (http://pyyaml.org/)
-   importlib (https://pypi.python.org/pypi/importlib/1.0.2) (only for python 2.6)

is required.
