# Sketcher CreateHexagon/ru

## Описание

Рисует шестиугольник, вписанный в круг вспомогательной геометрии. При запуске инструмента указатель мыши меняется на белый крест с красным значком шестиугольника. Координаты указателя отображаются рядом с ним синим цветом в режиме реального времени.

![](images/SketcherCreateHexagonExample.png‎ )

## Применение


<div class="mw-translate-fuzzy">

-   После нажатия кнопки **[<img src=images/Sketcher_CreateHexagon.png style="width:16px"> Шестиугольник**, нажмите первый раз, чтобы задать центр, затем переместите мышь и нажмите второй раз, чтобы задать одну из вершин.
-   Нажатие **Esc** или правой кнопки мыши закрывает функцию.


</div>

## Примечания

Видимая окружность является элементом вспомогательной геометрией, которая видна только при редактировании эскиза.
