# Part Ellipsoid/ru

## Описание

Команда <img alt="" src=images/Part_Ellipsoid.svg  style="width:24px;"> [Эллипсоид](Part_Ellipsoid/ru.md) создаёт параметрический твердотельный Эллипсоид.

Полученная форма ограничена в FreeCAD твёрдым (необязательно усечённым) сфероидом, формой, которую вы создали если бы, вращали эллипс вокруг одной из его осей. По умолчанию это [Сплюснутый сфероид](http://en.wikipedia.org/wiki/Oblate_spheroid), форма полученная вращением эллипса вокруг своей малой оси. Параметры могут быть изменены, чтобы сформировать [Вытянутый сфероид](http://en.wikipedia.org/wiki/Prolate_spheroid).

Сфероид по умолчанию в FreeCAD будет иметь круг для любого поперечного сечения, параллельного плоскости XY. Поперечное сечение, параллельное двум другим плоскостям, будет эллипсом.

В математике, [Эллипсоид](http://en.wikipedia.org/wiki/Ellipsoid) будет иметь эллиптическое поперечное сечение во всех трёх плоскостях.

## Свойства

-   Radius 1, по умолчанию малый радиус параллельный оси Z,
-   Radius 2, по умолчанию основной радиус, параллельный плоскости XY, также является максимальным радиусом круглого поперечного сечения
-   Angle 1, нижнее усечение эллипсоида, параллельное круглому поперечному сечению (-90 градусов в полном сфероиде)
-   Angle 2, верхнее усечение эллипсоида, параллельное круглому поперечному сечению (90 градусов в полном сфероиде)
-   Angle 3, угол поворота(охвата) эллиптического поперечного сечения (360 градусов в полном сфероиде)
