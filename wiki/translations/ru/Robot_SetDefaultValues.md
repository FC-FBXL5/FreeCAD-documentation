<div class="mw-translate-fuzzy">

#### Описание

Устанавливает значения по умолчанию для скорости, ускорения и целостности при создании промежуточных точек.


</div>

#### Использование

1.  Выберите в верхнем меню **Робот** → **<img src="images/Robot_SetDefaultValues.svg" width=32px>Установка значений по умолчанию**, чтобы установить   *

   *   

       *   \*скорость
       *   \*продолжительность (режим)
       *   \*ускорение


<div class="mw-translate-fuzzy">

#### Примечание

Программные значения по умолчанию следующие

-   скорость = 1 m/s
-   продолжительность = false
-   ускорение = 1 m/s\^2


</div>
