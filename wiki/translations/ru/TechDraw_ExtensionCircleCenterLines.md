## Описание

The <img alt="" src=images/TechDraw_ExtensionCircleCenterLines.svg  style="width:24px;"> **Arc-Circle Centerlines** tool adds centerlines to circles and arcs.

<img alt="" src=images/TechDraw_ExtensionCircleCenterLinesExample.png  style="width:200px;"> 
*On the right a circle with centerlines*

## Применение

1.  Optional pick favoured line attributes using **<img src="images/TechDraw_ExtensionSelectLineAttributes.svg" width=16px> [Aspect](TechDraw_ExtensionSelectLineAttributes.md)** button.
2.  Select one or more circles or arcs.
3.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_ExtensionCircleCenterLines.svg" width=16px> [Arc-Circle Centerlines](TechDraw_ExtensionCircleCenterLines.md)** button.
    -   Select the **TechDraw → Extensions: centerlines and threading → <img src="images/TechDraw_ExtensionCircleCenterLines.svg" width=16px> Draw circle centerlines** option from the menu.
4.  Centerlines are added to each circle or arc.

-   The added centerlines have the attributes (style, width, colour) selected using the **<img src="images/TechDraw_ExtensionSelectLineAttributes.svg" width=16px> [Aspect](TechDraw_ExtensionSelectLineAttributes.md)** button.
