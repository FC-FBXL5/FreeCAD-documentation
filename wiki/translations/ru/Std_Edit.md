## Описание

Команда **Редактировать/закончить редактирование** позволяет переключать объект в режим редактирования и обратно.

## Применение

1.  Если ни один объект не находится в режиме редактирования: выберите один объект.
2.  Выберите в меню параметр {{MenuCommand|Правка → <img src="images/Std_Edit.svg" width=16px> Редактировать/закончить редактирование}}.
3.  Режим редактирования выбранного объекта либо активируется, либо деактивируется существующий режим редактирования.

## Примечания


<div class="mw-translate-fuzzy">

-   Не все типы объектов имеют режим редактирования.
-   Функциональность, доступная в режиме редактирования, зависит от типа объекта.
-   Режим редактирования можно активировать дважды щелкнув объект в [древе проекта](Tree_view/ru.md).


</div>

## Программирование


**Смотрите так же:**

[Основы составления скриптов в FreeCAD](FreeCAD_Scripting_Basics/ru.md).

Для входа в режим редактирования объекта используйте метод `setEdit` объекта документа. Этот метод не доступен, когда FreeCAD в режиме консоли.


```python
import FreeCADGui

FreeCADGui.ActiveDocument.setEdit("myObjectName",0)
```

Второй аргумент - это режим редактирования. Доступны следующие опции:

0 = Default
1 = Transform
2 = Cutting
3 = Color

Для деактивации режима редактирования используйте метод `resetEdit` объекта document.


```python
import FreeCADGui

FreeCADGui.ActiveDocument.resetEdit()
```
