# Import Export Preferences/it
{{TOCright}}

## Introduzione

FreeCAD può importare ed esportare molti formati di file. Per alcuni formati esistono preferenze dedicate che si trovano nell\'[editor delle preferenze](Preferences_Editor/it.md), nel menu **Modifica → Preferenze → Import-Export**.

Non tutte le schede delle preferenze di importazione ed esportazione sono disponibili di default. Per alcune bisogna caricare prima un determinato ambiente di lavoro.

## Note

### Pagine TechDraw 

Le preferenze DXF e SVG elencate qui non vengono utilizzate dai comandi <img alt="" src=images/TechDraw_ExportPageSVG.svg  style="width:24px;"> [Esporta la pagina in SVG](TechDraw_ExportPageSVG/it.md) e <img alt="" src=images/TechDraw_ExportPageDXF.svg  style="width:24px;"> [Esporta la pagina in DXF](TechDraw_ExportPageDXF/it.md) dell\' [Ambiente TechDraw](TechDraw_Workbench/it.md)<img alt="" src=images/Workbench_TechDraw.svg  style="width:24px;">, o dall\'opzione di esportazione di [TechDraw](TechDraw_Workbench/it.md): **File → Esporta → Technical Drawing (*.svg *.dxf *.pdf)**.

### file OpenSCAD 

Le preferenze di importazione ed esportazione per i file OpenSCAD sono elencate in una posizione diversa nell\'[editor delle Preferenze](Preferences_Editor/it.md). Vedere le [Preferenze di OpenSCAD](OpenSCAD_Preferences/it.md).

### IGES

Il formato di file [Initial Graphics Exchange Specification](https://it.wikipedia.org/wiki/IGES) (IGES) è un formato di file che consente lo scambio digitale di informazioni tra i sistemi CAD. Dopo la pubblicazione del formato di file [STEP](Preferences_Editor#STEP.md), lo sviluppo di IGES è stato interrotto nel 1996, ma è ancora supportato da molti programmi CAD. I file IGES hanno l\'estensione **.iges** o **.igs**.

La scheda *IGES* viene visualizzata nelle preferenze solo se l\'[Ambiente Part](Part_Workbench/it.md)<img alt="" src=images/Workbench_Part.svg  style="width:24px;">, l\'[Ambiente PartDesign](PartDesign_Workbench/it.md)<img alt="" src=images/Workbench_PartDesign.svg  style="width:24px;"> o l\'[Ambiente OpenSCAD](OpenSCAD_Workbench/it.md)<img alt="" src=images/Workbench_OpenSCAD.svg  style="width:24px;"> è stato caricato nella sessione corrente di FreeCAD.

### IFC

[Industry Foundation Classes](http://en.wikipedia.org/wiki/Industry_Foundation_Classes) (IFC) è un formato ad ampia diffusione per lo scambio di dati tra le applicazioni [BIM](https://it.wikipedia.org/wiki/Building_Information_Modeling). È usato in architettura e ingegneria.

Nota per gli utenti Linux: per gestire questo formato di file FreeCAD richiede il [Modulo IfcOpenShell](Extra_python_modules/it.md).

### SVG

[Scalable Vector Graphics](SVG/it.md) (SVG) è un formato per [immagini vettoriali](https://it.wikipedia.org/wiki/Grafica_vettoriale) per la grafica bidimensionale. Un\'immagine vettoriale può essere ridimensionata a qualsiasi dimensione senza perdere forma o dettagli. Un\'immagine SVG può essere convertita in formati bitmap come PNG o JPEG per la stampa.

Per il formato SVG è possibile specificare quanto segue:

### VTK

Il [Visualization Toolkit](https://en.wikipedia.org/wiki/VTK) (VTK) è un sistema software open source disponibile gratuitamente per la computer grafica 3D, l\'elaborazione delle immagini e la visualizzazione. I file VTK vengono utilizzati dall\'[Ambiente FEM](FEM_Workbench/it.md) <img alt="" src=images/Workbench_FEM.svg  style="width:24px;"> per il [post processing](FEM_Post_Processing_based_on_VTK/it.md) dei risultati della simulazione.

La scheda *VTK* viene mostrata nelle preferenze solo se l\'[Ambiente FEM](FEM_Workbench/it.md) <img alt="" src=images/Workbench_FEM.svg  style="width:24px;"> è stato caricato nella sessione corrente di FreeCAD.

Per il formato VTK è possibile specificare quanto segue:

+++
| Nome                                       | Descrizione                                                                                                                                                                                                                                                                                  |
+============================================+==============================================================================================================================================================================================================================================================================================+
|                             | Seleziona quali oggetti devono essere importati.                                                                                                                                                                                                                                             |
| **Quale oggetto importare in** |                                                                                                                                                                                                                                                                                              |
|                                         | Se viene selezionato **Oggetto risultato VTK**, verrà importato un oggetto risultato VTK FEM di FreeCAD (uguale all\'oggetto che è stato esportato).                                                                                                                                         |
|                                            |                                                                                                                                                                                                                                                                                              |
|                                            | Se viene selezionato **Oggetto FEM mesh**, i risultati nel file VTK verranno omessi, verranno importati solo i dati mesh e verrà creato un oggetto mesh FEM di FreeCAD.                                                                                                                      |
|                                            |                                                                                                                                                                                                                                                                                              |
|                                            | Se viene selezionato **Oggetto risultato FreeCAD**, i dati importati verranno convertiti in un oggetto Risultato FEM di FreeCAD. **Nota:** questa impostazione richiede i nomi esatti dei componenti dei risultati e quindi funziona correttamente solo con i file VTK esportati da FreeCAD. |
+++
