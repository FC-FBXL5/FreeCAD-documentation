# Spreadsheet Import/it


</div>

## Descrizione


<div class="mw-translate-fuzzy">

Lo strumento <img alt="" src=images/Spreadsheet_Import.svg  style="width:16px;"> **Importa un foglio di calcolo** consente di importare un file con valori separati da virgola (CSV) in un foglio di calcolo in modo che possa essere modificato e utilizzato con le [espressioni](expressions/it.md).


</div>

## Utilizzo

1.  Premere il pulsante **[<img src=images/Spreadsheet_Import.svg style="width:16px"> Importa un foglio di calcolo**.
2.  Scegliere il file CSV.

## Notes

-   Use the [Std Import](Std_Import.md) command to import Excel ***.xlsx** files.
