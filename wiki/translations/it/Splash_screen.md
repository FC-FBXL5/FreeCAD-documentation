# Splash screen/it
Lo **Splash screen** è un\'immagine che appare durante l\'avvio di FreeCAD. È possibile disabilitare lo splash screen nel menu Preferenze di FreeCAD rimuovendo l\'opzione \"Abilita splash screen all\'avvio\".
