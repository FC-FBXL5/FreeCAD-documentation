# Spreadsheet CSV/it




<div class="mw-translate-fuzzy">

Il [formato CSV](http://en.wikipedia.org/wiki/Comma-separated_values) (Comma-Separated Values) è un formato di testo semplice per scambiare i dati dei fogli di calcolo. Di solito può essere importato ed esportato da qualsiasi applicazione foglio di calcolo come LibreOffice o Microsoft Excel. La lettura e la scrittura in formato csv è fatto dal [modulo csv](http://docs.python.org/2/library/csv.html) di Python interno.


</div>

## Importing


<div class="mw-translate-fuzzy">

## Importazione

Importando un file .csv file in FreeCAD si crea un oggetto [spreadsheet](Spreadsheet_Create/it.md) compilato con i valori del file.


</div>

## Exporting


<div class="mw-translate-fuzzy">

## Exportazione

Esportando un oggetto [spreadsheet](Spreadsheet_Create/it.md) in un file .csv file si scrivono semplicemente tutti i valori dell\'oggetto foglio di calcolo. Le funzioni, tipo \"=A3+B5\" sono scritte come funzioni, non come valori risultanti.


</div>
