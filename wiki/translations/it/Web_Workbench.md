## Introduzione


<div class="mw-translate-fuzzy">

L\'ambiente Web fornisce una finestra del browser all\'interno della vista 3D di FreeCAD, basata su [WebKit](https://webkit.org/).


</div>

Da questo browser è possibile navigare sul web, e accedere a siti web (i cookie sono supportati). L\'ambiente web è anche una base per sviluppare in FreeCAD degli strumenti web-correlati. Il browser web fornisce un paio di pulsanti di navigazione standard come stop, ricarica, pagina precedente e successiva.
