# TechDraw ExtensionCreateHorizChamferDimension/de

## Beschreibung

Das <img alt="" src=images/TechDraw_ExtensionCreateHorizChamferDimension.svg  style="width:24px;"> **TechDraw ExtensionCreateHorizChamferDimension** Werkzeug fügt eine horizontale Fasenbemaßung mit Winkelangabe ein.

<img alt="" src=images/TechDraw_ExtensionCreateHorizChamferDimensionExample.png  style="width:450px;"> 
*Rechts die erzeugte Bemaßung*

## Verwendung

1.  Wähle zwei Knotenpunkte.
2.  Es gibt mehrere Möglichkeiten das Werkzeug aufzurufen:
    -   Drücke die **<img src="images/TechDraw_ExtensionCreateHorizChamferDimension.svg" width=16px> [TechDraw ExtensionCreateHorizChamferDimension](TechDraw_ExtensionCreateHorizChamferDimension.md)** Schaltfläche.
    -   Wähle die **TechDraw → Extensions: Dimensions → <img src="images/TechDraw_ExtensionCreateHorizChamferDimension.svg" width=16px> Create Horizontal Chamfer Dimension** Option aus dem Menü.
3.  Eine Fasenbemaßung wird eingefügt.
