# PartDesign Draft/de


</div>

### Description


<div class="mw-translate-fuzzy">

### Beschreibung

Dieses Werkzeug erzeugt einen Winkelentwurf auf den ausgewählten Flächen eines Objekts. Ein neuer separater Eintrag für Entwürfe (gefolgt von einer fortlaufenden Nummer, wenn bereits Entwürfe im Dokument vorhanden sind) wird im Projektbaum erstellt.
