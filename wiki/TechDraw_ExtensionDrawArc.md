# TechDraw ExtensionDrawArc

## Description

The <img alt="" src=images/TechDraw_ExtensionDrawArc.svg  style="width:24px;"> **TechDraw ExtensionDrawArc** tool adds a cosmetic arc.

 <img alt="" src=images/TechDraw_ExtensionArcExample.png  style="width:400px;">  
*On the right an added cosmetic arc*

## Usage

1.  Optionally change the current line attributes with the <img alt="" src=images/TechDraw_ExtensionSelectLineAttributes.svg  style="width:16px;"> [TechDraw ExtensionSelectLineAttributes](TechDraw_ExtensionSelectLineAttributes.md) tool.
2.  Select the first vertex to determine the center of the arc.
3.  Select the second vertex to determine the radius and the start angle of the arc. The arc will be created counter clockwise.
4.  Select the third vertex to determine the end angle of the arc.
5.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_ExtensionDrawArc.svg" width=16px> [TechDraw ExtensionDrawArc](TechDraw_ExtensionDrawArc.md)** button.
    -   Select the **TechDraw → Extensions: Centerlines/Threading → <img src="images/TechDraw_ExtensionDrawArc.svg" width=16px> Add cosmetic arc** option from the menu.
6.  A cosmetic arc is added.
