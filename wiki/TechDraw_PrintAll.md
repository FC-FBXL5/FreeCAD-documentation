# TechDraw PrintAll

## Description

The <img alt="" src=images/TechDraw_PrintAll.svg  style="width   *24px;"> **TechDraw PrintAll** tool sends all the Pages in a document to the printer.

## Usage

1.  There are several ways to invoke the tool   *
    -   Press the **<img src="images/TechDraw_PrintAll.svg" width=16px> [Print All](TechDraw_PrintAll.md)** button.
    -   Select the **TechDraw → <img src="images/TechDraw_PrintAll.svg" width=16px> Print All** option from the menu.
    -   From the Right Mouse Button context menu.

## Scripting


**See also   ***

[TechDraw API](TechDraw_API.md) and [FreeCAD Scripting Basics](FreeCAD_Scripting_Basics.md).

The PrintAll tool is not yet available for scripting.

## Notes

PrintAll can be invoked from any window displaying a Page.
