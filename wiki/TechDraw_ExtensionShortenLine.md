# TechDraw ExtensionShortenLine

## Description

The <img alt="" src=images/TechDraw_ExtensionShortenLine.svg  style="width:24px;"> **TechDraw ExtensionShortenLine** tool shortens a line created with the [Extensions tools](TechDraw_Workbench#Extensions.md) at both ends.

 <img alt="" src=images/TechDraw_ExtensionShortenLineExample.png  style="width:300px;">  
*On the right a centerline shortened at both ends*

## Usage

1.  The delta distance is determined by \...?
2.  Select a single straight (center)line created with the [Extensions tools](TechDraw_Workbench#Extensions.md).
3.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_ExtensionShortenLine.svg" width=16px> [TechDraw ExtensionShortenLine](TechDraw_ExtensionShortenLine.md)** button.
    -   Select the **TechDraw → Extensions: Attributes/Modifications → <img src="images/TechDraw_ExtensionShortenLine.svg" width=16px> Shorten line** option from the menu.
4.  The line is shortened at both ends.
