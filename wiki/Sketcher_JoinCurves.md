# Sketcher JoinCurves

## Description

Joins two curves at selected end points.

## Usage

1.  Select\... then either:
    -   Press the button **[<img src=images/Sketcher_JoinCurves.svg style="width:16px"> [Join curves](Sketcher_JoinCurves.md)**.
    -   Use the menu **Sketch → Sketcher B-spline tools → [<img src=images/Sketcher_JoinCurves.svg style="width:16px"> Join curves**.
