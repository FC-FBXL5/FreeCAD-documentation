# Sketcher BSplineInsertKnot

## Description

Insert a knot into an existing B-spline (See [this page](B-Splines.md) for more info about B-splines). If a knot already exists at the specified parameter, increases the multiplicity of that knot by one.

## Usage

1.  Select a B-spline, then either:
    -   Press the button **[<img src=images/Sketcher_BSplineInsertKnot.svg style="width:24px"> [B-spline insert knot](Sketcher_BSplineInsertKnot.md)**.
    -   Use the menu **Sketch → Sketcher B-spline tools → [<img src=images/Sketcher_BSplineInsertKnot.svg style="width:24px"> Insert knot**.
2.  Enter the parameter for the new knot. The existing knots and their multiplicities are provided for reference.
3.  (Optionally) Use <img alt="" src=images/Sketcher_BSplineIncreaseKnotMultiplicity.svg  style="width:24px;"> [Increase knot multiplicity](Sketcher_BSplineIncreaseKnotMultiplicity.md) if a knot with a higher multiplicity is desired.
